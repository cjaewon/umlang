## 📚 문법
```
[정수 ( 아직 까지 음수 지원 X )]
"음" : 1
"." : 0

"." 을 0, "음" 을 1 로 해서 숫자를 이진수로 표현한다.
단, "음" 바로 앞에 붙은 "." 은 없는 문자 취급한다.
(언어의 컨셉을 유지하기 위함)

ex) 음..음.. // 1010 -> 10
(음..음.. 이 1010 인 이유는 음 앞에 있는 "."한개을 무시하기 때문인다. ) 
ex) 음음..음 // 1101 -> 13

[사칙 연산]

"?" : 더하기 연산
"!" : 빼기 연산
"??" : 곱하기 연산
"!!" : 나누기 연산
"?!" : 나머지 연산

ex) 음.음.?음음..음 // 35

[메모리]
";" : 스택에 집어넣기

숫자를 다 만들었다면, ";" 을 통해 스택에 집어넣는다.
";" 가 2개 이상이라면, 그만큼의 똑같은 수를 집어넣는다.

ex) 음..;; // 4 두개를 넣는다. <스택>[4, 4]
```
